# Changelog

All notable changes to VUTOOLS will be documented in this file.

## [2.0.0] - 2025-12-08

### Added
- 30+ new Discord bot commands
- Group Scanner tool for finding claimable Roblox groups
- Server Lookup tool for Discord server information
- Asset Downloader for Roblox assets
- User ID Lookup for Discord users
- Game Tracker with real-time statistics
- Enhanced analytics dashboard
- Multiple new themes
- Auto-update system

### Changed
- Improved UI/UX across all tools
- Better error handling and user feedback
- Optimized performance and memory usage
- Updated Discord.js to v14
- Modernized codebase

### Fixed
- Asset downloader showing incorrect data
- Game tracker not updating properly
- Various UI glitches
- Memory leaks in long sessions
