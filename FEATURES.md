# What's in VUTOOLS

Here's everything the app can do. No fluff, just the list.

## Roblox Stuff

**Usernames**
- Check if usernames are available
- Snipe usernames when they become available
- Generate random usernames
- Track username changes

**Users**
- Look up any user's profile
- Check if two users are friends
- Analyze avatars and items
- View stats and badges

**Groups**
- Get group info
- View member lists
- Scan for claimable groups (finds groups with no owner)

**Games**
- Search for games
- See what's trending
- Get detailed game info
- Track game stats in real-time (players, visits, favorites)

**Assets**
- Download asset info and links
- View asset details
- Search the catalog

## Discord Stuff

**Bot Commands (41 total)**

Moderation - ban, kick, mute, unmute, warn, clear messages, slowmode, lock/unlock channels, nuke, wipeout

Server Management - announcements, custom embeds, polls, giveaways, roles, nicknames, avatars, server info, user info

Fun - 8ball, coinflip, dice, memes, jokes, quotes, facts, roasts, compliments

Utilities - reminders, calculator, weather, translate, URL shortener, QR codes, timestamps

Info - ping, bot info, server list, help

**Other Discord Tools**
- Webhook manager
- Server lookup (paste invite codes)
- User ID lookup (shows account age)

## Trading Tools

- Trade calculator
- Limited item tracker

- Value analyzer
- Price history
- Market trends
- Deal finder

## Analytics

- Dashboard with everything
- History of what you've done
- Found list (usernames, items, etc.)
- Trends and stats

## Utilities

- Import/export data (JSON, CSV, TXT)

- Auto-updates
- Settings and customization

Everyone gets access to all tools with a valid key. No tiers, no premium BS. You get everything.
